# **App Name**: Grafana and Friends Mumbai

## Core Features:

- Event Information Display: Display event details like date, time, venue (tbd) with Google Maps integration), and registration link to Konfhub (https://konfhub.com/gafm26#tickets).
- Speaker Showcase: Showcase event speakers with details and a CFP (Call for Proposals) link.
- Event Schedule: Present a detailed event schedule with time slots (10AM-5PM) and activities.
- Sponsor Section: Display sponsors in tiered sections (Platinum: Grafana Labs, Silver/Bronze placeholders) with logos and links.
- Past Events Gallery: Show a gallery of images from past Grafana and Friends Mumbai events.
- FAQ Section: Implement an accordion-style FAQ section to answer common questions.

## Style Guidelines:

- Primary color: Deep orange (#E64A19), reminiscent of Mumbai sunsets, brings warmth and energy to the design.
- Background color: Light desaturated orange (#FAF2EC) offers a gentle backdrop that enhances readability.
- Accent color: Analogous red-orange (#FF5722), slightly brighter, used for call-to-action buttons and important interactive elements, adding contrast and capturing the city's vibrancy.
- Font pairing: 'Poppins' (sans-serif) for headers and 'PT Sans' (sans-serif) for body text.
- Use simple, modern icons related to Grafana, observability, and the Mumbai community.
- Single-page layout with smooth scrolling for intuitive navigation, similar to the reference site.
- Parallax scrolling effect for the hero section and subtle animations for interactive elements to enhance user engagement.
